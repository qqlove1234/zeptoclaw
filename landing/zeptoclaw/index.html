The OpenClaw Family

One Vision, Four Languages

It started with OpenClaw ‚Äî a TypeScript powerhouse with 52+ modules and 12 channels. It could do everything. But "everything" comes at a cost: complexity, dependencies, and resource bloat.

Then came NanoClaw ‚Äî a forkable, understandable assistant in ~5,000 lines of TypeScript. Then PicoClaw pushed further ‚Äî a Go binary that runs on a $10 RISC-V board with under 10MB RAM.

ZeptoClaw is the final form: Rust's memory safety, async performance, and container isolation ‚Äî built for teams who need security and multi-tenancy without sacrificing simplicity.

Getting Started

Up and Running in 60 Seconds

1. Build from source

$ git clone https://github.com/qhkm/zeptoclaw.git
$ cd zeptoclaw && cargo build --release

2. Run interactive setup

$ ./target/release/zeptoclaw onboard
# Walks you through provider keys, channels, and workspace config

3. Start your assistant

# CLI mode with streaming
$ zeptoclaw agent --stream -m "Hello, set up my workspace"

# Use a built-in template
$ zeptoclaw agent --template researcher -m "Analyze this API"

# Gateway mode ‚Äî Telegram/Slack/Discord/Webhook
$ zeptoclaw gateway

# Batch mode ‚Äî process prompts from file
$ zeptoclaw batch --input prompts.txt --format jsonl
